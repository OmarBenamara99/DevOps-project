# DevSecOps CI/CD Pipeline for Spring Boot & Angular Application

## Overview
Automated the full lifecycle of a Spring Boot and Angular application, ensuring **quality, security, and monitoring** throughout development and deployment.

## Tech Stack
- **Backend:** Spring Boot, Maven  
- **Frontend:** Angular  
- **CI/CD Tools:** Jenkins, GitHub Actions, Docker, Docker Compose, Nexus  
- **Security & Quality:** SonarQube, Snyk, Trivy  
- **Monitoring:** Prometheus, Grafana  

## Key Features
- Automates build, test, and deployment for backend and frontend  
- Ensures code quality and security through automated scans  
- Deploys the application in production with Docker Compose  
- Monitors application performance and health with Prometheus and Grafana  

## Achievements
- Streamlined the full CI/CD process for a multi-tier application  
- Enhanced **security and reliability** across the development lifecycle  

## What I Learned
- Building and automating CI/CD pipelines for full-stack applications  
- Integrating security scanning and monitoring into DevOps workflows
